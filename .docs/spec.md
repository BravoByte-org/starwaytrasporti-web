# Starway Trasporti — Project Spec

## Overview

StarwayTrasporti.com is an international transportation website that serves to highlight an exceptional service, generate new clients, and help existing clients find the information they need.

## Status

Pre-production. First BravoByte delivery repo and first extraction source for shared business-core modules.

## Goals

1. Launch MVP website for Starway Trasporti
2. Serve as the first consumer of BravoByte shared packages
3. Prove the delivery-repo-consumes-core-repos pattern
4. Establish bilingual content strategy (Italian/English)
