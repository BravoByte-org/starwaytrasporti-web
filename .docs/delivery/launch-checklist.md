# Launch Checklist

## Pre-launch

- [ ] All MVP pages implemented and content-populated
- [ ] CMS content complete (Italian and English)
- [ ] SEO metadata configured for all pages
- [ ] Performance audit passed (Lighthouse)
- [ ] Accessibility audit passed
- [ ] E2E tests passing
- [ ] Unit tests passing
- [ ] Environment variables configured in Vercel production
- [ ] Domain DNS configured
- [ ] SSL certificate active
- [ ] Error pages implemented (`+error.svelte`)
- [ ] Analytics configured
- [ ] Favicon and social share images set

## Post-launch

- [ ] Verify production deployment
- [ ] Monitor error rates
- [ ] Verify CMS content renders correctly
- [ ] Check all external links
- [ ] Confirm bilingual content displays correctly
