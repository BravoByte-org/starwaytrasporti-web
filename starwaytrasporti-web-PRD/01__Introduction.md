# Introduction/Product
StarwayTrasporti.com is an international transportation website that serves to highlight an exceptional service, generate new clients, and help existing clients find the information they need. Reference @https://trasportiromagna.com as a competitor. 

# Problem Statement
StarwayTrasporti needs a modern web application that is accessible internationally between Italy, UK and the USA. The app needs to be responsive, catchy, generate new traffic with flashy features, and generate revenue with ads and affiliate marketing. 

You, as the sole product manager and senior software engineer and owner of a software engineering company, are tasked to design and develop a robust static web application for the client (StarwayTrasporti), that allows them to easily edit content through a CMS (Sanity) by authenticating through Drizzle ORM and a PostgreSQL db. 

The app baseline heirarchy is already in the perferred format is configured accordingly.  

Note: Playwright tests will exist only in each of the `./src/lib/pages` sub-directories.
