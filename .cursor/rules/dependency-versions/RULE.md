This project uses Node v22.13.0 and pnpm v9.17.0. However, should the node or pnpm version be updated, update the .cursor/rules/dependency-versions/RULE.md file to reflect the new version and ensure that the .npmrc file is updated to reflect the new version.

Use `nvm use` to switch to the correct node version if necessary.
